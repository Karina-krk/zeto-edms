# Zeto EDMS Landing

Статический многостраничный сайт-презентация системы Zeto EDMS.

## О проекте

Сайт состоит из двух основных страниц:

- Главная (`index.html`) с hero-блоком, карточками этапов проекта, блоком клиентов и контактами.
- Страница «О нас» (`about.html`) с описанием системы и команды.

Также в репозитории есть черновая страница `projects.html`.

## Функциональность

- Переключение языка интерфейса: RU/EN.
- Сохранение выбранного языка в `localStorage`.
- Боковое меню (sidebar) для мобильной и десктопной навигации.
- Анимации появления блоков при скролле (`fade-in`).
- Раскрытие дополнительного текста в карточках на главной странице.
- Кнопка перехода в trial-версию Zeto EDMS.

## Технологии

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Локальные шрифты из папки `fonts/`

## Структура проекта

```text
.
├── index.html         # главная страница
├── about.html         # страница «О нас»
├── projects.html      # черновая страница проектов
├── style.css          # общие стили
├── home.css           # стили главной
├── about.css          # стили страницы «О нас»
├── script.js          # логика UI, i18n, меню, анимации
├── pict/              # изображения и медиа
└── fonts/             # локальные шрифты
```

## Локальный запуск

Проект не требует сборки и зависимостей.

1. Откройте папку проекта в VS Code.
2. Запустите локальный сервер (например, Live Server) и откройте `index.html`.

Альтернатива через Python:

```bash
python3 -m http.server 5500
```

После запуска сайт будет доступен по адресу:

`http://localhost:5500`
